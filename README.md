#97share discuss
##开发说明
本网站为.net课程设计所作。――2013年6月25日
##开发环境
基于.Net Framework 4.5，visual studio 2012 update 3开发。
数据库为SQL Server Management Studio 2008-R2
##简介
"97share"意味着"就去分享"～～
论坛系统为人与人之间的交流提供了一种更为便捷的方式，更适合于生活节奏加快、信息蕴含量更为丰富的现代社会。
论坛不仅广泛应用于社会生活领域，同时在商业交流、协作学习方面产生了极大的影响力。
用户可以发表自己的言论，其他用户可以对贴子进行评论（跟帖）或发表更多言论（发帖）。
##注意
导入数据库之后。请修改Web.config文件信息：
connectionString="server=what-pc;database=forum;uid=what;pwd=what"
